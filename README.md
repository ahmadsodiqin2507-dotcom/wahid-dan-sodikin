# Koperasi Pinjaman Online

Website koperasi pinjaman online dengan fitur admin dan peminjam.

## Fitur
- Login akun admin dan peminjam
- Dashboard admin untuk melihat pengajuan pinjaman dan status anggota
- Dashboard peminjam untuk mengajukan pinjaman baru
- Tampilan profesional dan responsif

## Akun Contoh
- Admin: `admin@koperasi.com` / `Admin123!`
- Peminjam: `user@koperasi.com` / `Borrower123!`

## Cara Menjalankan
Cukup buka `index.html` di browser, atau jalankan server lokal statis jika diperlukan.

## Struktur File
- `index.html` - halaman utama dan login
- `styles.css` - gaya tampilan profesional
- `app.js` - logika login dan dashboard
